# Amazon-Sales-Dashboard
Developed an interactive Amazon Sales Dashboard using Power BI and Excel to analyze sales, profit, orders, and customer trends. Cleaned data with Power Query, created DAX measures, and built KPI cards, charts, and slicers for dynamic analysis by category, region, and time. Delivered actionable insights to support data-driven business decisions.


Project Description
Amazon Sales Dashboard | Power BI, Excel

Developed an interactive Amazon Sales Dashboard in Power BI to analyze sales performance, customer behavior, and business trends. The dashboard provides insights into key performance indicators (KPIs) such as total sales, profit, orders, quantity sold, and average order value. It includes dynamic filters and visualizations that help users analyze sales by product category, region, state, customer segment, and time period for better business decision-making.

Analyzed sales by:
